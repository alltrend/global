# Digital Clock - Multiple Time Zones

A beautiful digital clock web application that displays the current time across 8 major time zones around the world.

## Features

✨ **Multiple Time Zones**: Display current time in 8 major cities:
- New York (EST)
- London (GMT)
- Tokyo (JST)
- Sydney (AEDT)
- Dubai (GST)
- Singapore (SGT)
- Los Angeles (PST)
- Hong Kong (HKT)

✨ **Real-Time Updates**: Time updates every second

✨ **Local Time Display**: Shows your local time and date

✨ **Beautiful UI**: 
- Modern gradient background
- Smooth animations
- Responsive design
- Hover effects on clock cards

✨ **Responsive**: Works perfectly on desktop, tablet, and mobile devices

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Watch the clocks update in real-time!

## Technologies Used

- **HTML5**: Structure
- **CSS3**: Styling with animations and gradients
- **JavaScript (Vanilla)**: Time zone calculations and real-time updates

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling and animations
- `script.js` - JavaScript logic for time calculations
- `README.md` - Documentation

## How It Works

The application uses JavaScript to:
1. Get the current UTC time
2. Calculate the time for each time zone based on UTC offset
3. Format and display the time in HH:MM:SS format
4. Update every second to show real-time changes
5. Display your local time and date
